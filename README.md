## Installation du projet

1. clone du projet
2. changement des variables dans le `.env`
3. lancer la commande `composer install`
4. lancer la commande `php -S 127.0.0.1:8000`
5. se rendre sur [127.0.0.1:8000/user](http://127.0.0.1:8000/user)
